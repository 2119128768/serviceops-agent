# Escalation Policy

Route tickets by issue and risk:

- API errors and latency: platform technical support
- Recharge, quota sync, invoice, refund: billing system group
- Deployment failures: model serving group
- RAG import and retrieval quality: knowledge platform group
- Permission and audit issues: security operations group

P0 and P1 incidents require immediate incident commander notification. P2 issues require response within four business hours.
